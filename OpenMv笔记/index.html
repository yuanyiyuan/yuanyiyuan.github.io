一. 感光元件 <核心>：

import sensor    //感光元件模块 ---> 用于设置感光元件的参数
sensor.reset()   //初始化感光元件
sensor.set_pixformat()  //设置像素模式
    |-->  sensor.GRAYSCALE //灰度，每个像素8bit
    |-->  sensor.RGB565    //彩色，每个像素16bit
sensor.set_framesize()  //设置图像的大小
    常见的图像大小： VGA: 640x480 | QVGA: 320x240 ...  SIF: 352x240 | QSIF: 176x120 ... CIF: 352x288 ...前面每加一个Q ，各个值除以2
sensor.skip_frames(n)  //跳过n张照片，在更改设置后，跳过一些帧，等待感光元件变稳定

!! sensor.snapshot() //拍摄一张照片，返回一个image对象
sensor.set_auto_gain() //自动增益开启（True）或者关闭（False） [在使用颜色追踪时，需要关闭自动增益]
sensor.set_auto_whitebal() //自动白平衡开启（True）或者关闭（False） [在使用颜色追踪时，需要关闭自动白平衡]
!! sensor.set_auto_exposure(enable[, exposure_us])
|–> enable 打开（True）或关闭（False）自动曝光 (默认打开)
|–> 如果 enable 为False，则可以用 exposure_us 设置一个固定的曝光时间（以微秒为单位）
sensor.set_windowing(roi) //处理图像中要提取出的处理的区域
|–> roi 的格式是 (x,y,w,h)
|–>
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~·
x:ROI区域中左上角的x坐标
y:ROI区域中左上角的y坐标
w:ROI的宽度
h:ROI的高度
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~·
sensor.set_hmirror(True) //水平方向翻转
sensor.set_vflip(True) //垂直方向翻转

二. 图像信息的返回

—–像素点
!! image.get_pixel(x,y) //获取一个像素点的值
|–> 对于灰度图: 返回(x,y)坐标的灰度值 / 对于彩色图: 返回(x,y)坐标的(r,g,b)的tuple
!! image.set_pixel(x, y, pixel) //设置一个像素点的值
|–> 对于灰度图: 设置(x,y)坐标的灰度值 / 对于彩色图: 设置(x,y)坐标的(r,g,b)的值
—–图像
!! image.width() //返回图像的宽度(像素)
!! image.height() //返回图像的高度(像素)
image.format() //灰度图会返回 sensor.GRAYSCALE，彩色图会返回 sensor.RGB565
image.size() //返回图像的大小(byte)
—–图像运算
image.invert() //取反，对于二值化的图像，0(黑)变成1(白)，1(白)变成0(黑)
image.nand(image) //与另一个图片进行与非（NAND）运算
image.nor(image) //与另一个图片进行或非（NOR）运算
xor –> 异或 xnor –> 异或非
image.difference(image) //从这张图片减去另一个图片 [用来做移动检测]

    平均数，中位数，众数，标准差，最小值，最大值，第一四分数，第三四分数
    mean   median  mode  stdev   min    max       lq         uq
image.get_statistics()   //统计区域内的图像信息

    statistics=img.get_statistics(roi=ROI)

三. 画图
画线： image.draw_line(line_tuple, color=White) //在图像中画一条直线
|–> line_tuple的格式是(x0, y0, x1, y1)，意思是(x0, y0)到(x1, y1)的直线
|–> color 可以是灰度值(0-255)，或者是彩色值(r, g, b)的tupple
画框： image.draw_rectangle(rect_tuple, color=White) //在图像中画一个矩形框
|–> rect_tuple 的格式是 (x, y, w, h)
画圆： image.draw_circle(x, y, radius, color=White) //在图像中画一个圆
|–> x,y是圆心坐标 / radius是圆的半径
画十字： image.draw_cross(x, y, size=5, color=White) //在图像中画一个十字
|–> x,y是坐标 / size是两侧的尺寸
写字： image.draw_string(x, y, text, color=White) //在图像中写字 8x10的像素
|–> x,y是坐标。使用\n, \r, and \r\n会使光标移动到下一行
|–> text是要写的字符串

四. 色块的选择
image.find_blobs(thresholds, // thresholds是颜色的阈值 –> 此参数是一个列表，可以包含多个颜色
roi=Auto, // roi为要提取的区域
x_stride=2, // 要查找捕捉的色块的x方向上最小宽度的像素 <可用来筛选所需的物体>
y_stride=1, // 要查找捕捉的色块的y方向上最小宽度的像素 <可用来筛选所需的物体>
invert=False, // 反转阈值: 把阈值以外的颜色作为阈值进行查找
area_threshold=10, // 面积阈值: 如果色块被框起来的面积小于这个值，会被过滤掉
pixels_threshold=10, // 像素个数阈值，如果色块像素数量小于这个值，会被过滤掉
merge=False, // 合并 [会合并所有的blob,无论是什么颜色]
margin=0, // 边界 [如果间距为0一个像素点，也会被合并]
threshold_cb=None,
merge_cb=None)
在返回的色块对象blob可以调用code方法，来判断是什么颜色的色块
eg:
img=sensor.snapshot()
red_blobs = img.find_blobs([red])
一个颜色阈值的结构: xxx = (minL, maxL, minA, maxA, minB, maxB)


五. 图像常用功能
模板匹配（find_template）:
|-> 采用的是ncc算法，只能匹配与模板图片大小和角度基本一致的图案
首先：加载模板图片 –> template = image.Image(“/xxx.pgm”) // 模板图片为pgm
其次：图片与模板图片对比 –> find_template(template, threshold, [roi, step, search])
eg:
img.find_template(template, 0.70, roi=(10,0,60,60), step=4, search=SEARCH_EX)
|–> threshold的0.7是相似度阈值
|–> roi是进行匹配的区域（左上顶点为（10，0），长60宽60的矩形） [roi的大小要比模板图片大]
特征点检测（find_keypoint）: 首先例程提取最开始的图像作为目标物体特征，然后kpts保存目标物体的特征 —> 默认会匹配目标特征的多种比例大小和角度
eg:
kpts = None //定义一个kpts保存目标物体的特征
// kpts1 = image.load_descriptor(“/desc.orb”) //导入特征【不建议】
kpts = img.find_keypoints(max_keypoints=150, threshold=10, scale_factor=1.2)
// image.find_keypoints(roi=Auto, threshold=20, normalized=False, scale_factor=1.5, max_keypoints=100, corner_detector=CORNER_AGAST) 原函数
|–> roi表示识别的区域，是一个元组（x,y,w,h）,默认与framsesize大小一致
|–> threshold是0~255的一个阈值，用来控制特征点检测的角点数量 <阈值越低，获得的角点越多>
|–> normalized是一个布尔数值，默认是False，可以匹配目标特征的多种大小（比ncc模版匹配效果灵活）。如果设置为True，关闭特征点检测的多比例结果，仅匹配
| 目标特征的一种大小（类似于模版匹配），但是运算速度会更快一些
|–> scale_factor是一个大于1.0的浮点数。这个数值越高，检测速度越快，但是匹配准确率会下降
|–> max_keypoints是一个物体可提取的特征点的最大数量
|–> corner_detector是特征点检测采取的算法，默认是AGAST算法, FAST算法会更快但是准确率会下降
match = image.match_descriptor(kpts, kpts2, threshold=85) //kpt和kpt2的匹配的近似特征点数目
// image.match_descriptor(descritor0, descriptor1, threshold=70, filter_outliers=False)。本函数返回kptmatch对象
|–> threshold阈值设置匹配的准确度，用来过滤掉有歧义的匹配。这个值越小，准确度越高
|–> filter_outliers默认关闭
测量距离： 距离 = 一个常数/直径的像素 –> [实际长度和摄像头里的像素成反比]

        eg:     length = K/Lm     //Lm 为直径的像素
扫码：    img.lens_corr   //来矫正 QR码、二维码的畸变
         img.find_barcodes()    //检测出条形码的信息
         img.find_qrcodes()     //检测出二维码的信息
保存截取到的图片到SD卡：
        sensor.snapshot().save("singtown/s%s/%s.pgm" % (num, n) )    //# or "example.bmp" (or others)
            // 先在OpenMV的U盘里新建一个文件夹，命名为singtown
检测人脸：
            d0 = img.find_lbp((0, 0, img.width(), img.height()))
            // d0为当前人脸的lbp特征
计算帧率： clock.fps()

from pyb import Timer
tim = Timer(4, freq=1000) #定时器4，频率为1000Hz —> 使用定时器4创建定时器对象-以1000Hz触发 [1000Hz为总数]
tim.counter() # 获取定时器的值
tim.freq(0.5) # 0.5 Hz
tim.callback(lambda t: pyb.LED(1).toggle()) # 回调函数，回调设置为 lambda t: pyb.LED(1).toggle() 函数

        spi = SPI(2, SPI.MASTER, baudrate=200000, polarity=1, phase=0)       #baudrate:波特率
        spi.send('hello')
        spi.recv(5)       # receive 5 bytes on the bus
        spi.send_recv('hello')      # send a receive 5 bytes
    I2C:
        from machine import I2C, Pin
        i2c = I2C(sda=Pin('P5'),scl=Pin('P4'))

        i2c.scan()
        i2c.writeto(0x42, b'123')         # write 3 bytes to slave with 7-bit address 42
        i2c.readfrom(0x42, 4)             # read 4 bytes from slave with 7-bit address 42

        i2c.readfrom_mem(0x42, 8, 3)      # read 3 bytes from memory of slave 42,
                                        # starting at memory-address 8 in the slave
        i2c.writeto_mem(0x42, 2, b'\x10') # write 1 byte to memory of slave 42
                                        # starting at address 2 in the slave
    Servo(舵机)：

        s1 = Servo(1)        # servo on position 1 (P7)       Servo(1) -> P7 (PD12)  |  Servo(2) -> P8 (PD13)
        s1.angle(45)         # move to 45 degrees
        s1.angle(-60, 1500)  # move to -60 degrees in 1500ms
        s1.speed(50)         # for continuous rotation servos
