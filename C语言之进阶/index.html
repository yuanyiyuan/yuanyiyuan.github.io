C语言之进阶

一. 时间函数 (介绍C语言中与时间有关的系统函数)
返回时间函数 time :
time_t time (time_t *t); //参数t是一个指针，如果不是一个空指针，则会将返回值存到t指针所指的内存单元中
//如果t是一个空指针，函数会返回一个time_t型长整型数


取当前时间函数 gmtime : 将time_t表示的秒数转换为人可以理解的时间
struct tm *gmtime(time_t *timep); //返回值为一个tm类型的结构体指针

字符串格式时间函数 ctime : 将一个时间返回成一个可以识别的字符串格式
char *ctime(time_t *timep);
取当地时间的函数 localtime :
struct tm *localtime(time_t *timep);
将时间转换成秒数的函数 mktime ：
time_t mktime(tm *timeptr);

二. 目录与文件
创建目录函数 mkdir :
int mkdir(char *pathname,mode_t mode); //pathname 是一个字符串，表示需要创建的目录路径 mode是表示权限的八进制数字
// 返回整型数 0 —> 代表 目录创建成功 否则返回 -1
删除目录函数 rmdir :
int rmdir(char *pathname); //pathname 是一个需要删除的目录字符串指针
创建文件函数 creat :
int creat(char *pathname, mode_t mode); //pathname 是表示需要建立文件的文件名和目录名 mode是表示权限
删除文件函数 remove :
int remove(char *pathname);
临时建立文件函数 mkstemp :
int mkstemp(char *template); //template是一个文件的文件名和目录名的字符串
