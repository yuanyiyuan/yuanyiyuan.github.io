一. 基础__1

*vim 文本编译器
vim 命令
格式： vim 文件名
vim 文本编译器 ： 有3种工作模式 ———— 可视模式、编辑模式、命令模式
可视模式 [默认模式] : 只能查看内容，不能修改内容
编辑模式 : 即可查看也可修改
命令模式 : 可给vim发送控制命令 ，此模式也是能看不能修改

可视模式 的命令 ：
    移动光标：
    <移动一个字符>------        h(左移),  j(下移),  k(上移),  l(右移)        
    <移动行>------        $ (移动到当前行尾)   0 (移动到当前行首)   : (移动到指定的行号)   G (移动到最后一行)   gg (移动到第一行)
    <页滚动>------        Ctrl+u (使文本向上滚动半屏)  Ctrl+e (使文本向上滚动一行)  Ctrl+y (使文本向下滚动一行)
                         Ctrl+f (使文本向上滚动一整屏)   Ctrl+b (使文本向下滚动一整屏)   zz(把光标所在行滚动到屏幕正中间)

    行选中 ：  shift + v  然后方向键选中
    列选中 ：  ctrl + v  然后方向键选中
    选中某个单词 ： 先按 v 然后左右键选中
    复制 ： y
    粘贴 ： p
    剪切 ： x
    撤销 ： u
    撤销后反悔 ：ctrl + r

    单个字母替换 ： 先按R ，然后输入新的字母
    自动补全 ：ctrl + n     [输入一个单词开头即可]

    高亮显示 ： 先输入 / 然后跟要高亮的单词
    ***屏幕之间的切换用快捷键 ： ctrl+ww (可视模式下)
命令行模式 的命令 ：    
    去除高亮 ： 命令行输入 ： :nohl
    w : 保存文件    q : 不保存退出    wq : 保存并退出    q! : 强制不保存退出
    %s/老信息/新信息/g : 字符串 ，单词替换
    直接输入行号 ： 直接跳转到第几行
    vs  文件名  ： 左右分屏
    sp  文件名  ： 上下分屏

二. 基础__2

shell 编程篇
Linux常用命令：
1.目录与文件命令：
查看目录命令(获取当前所在的绝对路径) pwd
切换目录命令(路径切换命令) cd
cd/ <切换到根目录> cd.. <切换到当前目录的上一级目录> cd~ <切换到用户的主目录>
查看目录命令(查看目录下的目录 和 文件的详细信息命令) ls
ls -a <显示所有文件及目录，包括隐藏文件> ls -lh <显示文件的详细列表，文件名，权限等信息> ls -R <递归列出目录下的所有文件>


** Tab 键 ：补齐对应的文件名 **
创建空文件的命令 touch 格式 ： touch 文件名
创建空目录的命令 mkdir 格式 ： mkdir 新目录名
删除文件和目录命令 rm 格式 ： rm -fr 目录名或者文件名
拷贝目录和文件命令 cp 格式 ： cp -fr 源文件/源目录 目标文件/目标目录
剪切/移动/重命名命令mv 格式 ： mv 源文件/源目录 目标文件/目标目录
创建快捷方式/链接命令ln 格式 ： ln -s //建立符号链接(快捷方式图案)
查看文件信息命令 wc -l :文件的文本行数 -w :文件的字数 -c :文件中的字符数
文件查看命令 cat 格式 ： cat 文件名
打印命令 echo [一般配合’ > ‘或者 ‘ >> ‘使用]
echo 打印信息 > 文件名 【会把原先的内容给覆盖】
echo 打印信息 >> 文件名 【会在原先的内容后添加】
查找文件命令 find 格式： find 路径 -name 文件名
在文件中搜索字符串信息命令 grep 格式： grep -Rn “要搜索的信息” 文件名
查看当前登录Linux系统的用户名 whoami
文件权限设置命令 chmod + 表示增加权限 - 表示取消权限 = 表示唯一设定权限

        linux系统打包解包命令   tar      //（类似rar格式的压缩包）
                打包
                tar -jcvf  压缩包名.tar.bz2  目录或者文件             【把目录和文件打包成bz2这种压缩格式】
                tar -zcvf  压缩包名.tar.gz   目录或者文件             【把目录和文件打包成 gz 这种压缩格式】
                解包
                tar -xvf  压缩包名.tar.bz2 / 压缩包名.tar.gz   
    2.man 系统帮助命令
        man命令    默认显示为命令手册
shell程序
    shell语言是一种编译型语言，编译时无需事先声明变量，对一个变量赋值时即定义了变量
    在shell程序中，“#”后面的内容为注释信息
    shell程序基本结构：
            所有shell程序的第一行都是以 #！开头， 后面跟上执行shell程序的shell解释器完整路径
            第二行以注释的方式写出程序的名称
        **shell程序为一个普通的文本文件，需要添加可执行权限后才可以执行**
    shell中的变量可分为 局部变量、环境变量、位置变量
    shell程序中无数据类型，变量赋值后只需在变量前加$即可访问变量的值
        环境变量的查看： export命令
        访问环境变量  ： 在环境变量的前面加上一个$符号
        定义环境变量  ： export命令来定义一个环境变量

        位置变量： 指shell程序在运行时传入的参数

        shell的变量赋值、算术运算符  == C语言
        shell的输入和输出：
            echo 命令输出结果
            read 命令读取信息 （可从键盘或文件中读入信息，并赋给一个变量）

        测试语句：指对变量的大小、字符串、文件属性等内容进行判断   test
            test 对文件测试 ： -d 测试文件是否是目录文件         -s  测试文件是否非空       -w 测试文件是否可写      -x  测试文件是否可执行
            test 对数值测试 ： -eq 两数是否相等    -ne 两数是否不相等   -le 第一个数是否不大于第二个数   -ge  第一个数是否不小于第二个数
                             -gt  第一个数是否大于第二个数   -gt  第一个数是否小于第二个数     [[test 3 -eq 5]]
            test 对字符串测试：  =  是否相等   ！= 是否不相等    -z 测试字符串是否是空字符   -n 测试字符串是非否是空字符
            test 对逻辑测试  ： -a :逻辑与   -o :逻辑或   ! :逻辑非
        shell 流程控制语句：
            if语句：
                if 条件1
                then 命令1
                elif 条件2
                then 命令2
                else 命令3
                fi
            for语句：
                for 变量名i  n 列表
                do
                命令1
                命令2...
                done
            until语句：      用于执行一个循环体，直到条件为真时停止   [[[与平常条件判断语句不一样，这个条件为真停止循环]]]
                until 条件
                do
                命令1
                ...
                done
