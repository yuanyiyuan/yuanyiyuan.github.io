arduino常见的库

IO库
    pinMode()   引脚的配置
        |-->第一参数：pin 引脚号
        |-->第二参数：INPUT  /  OUTPUT   [输入/输出]-->【默认为输入，引脚为高阻态】
                |-->  若输入态，可为 INPUT_PULLUP ：输入且为内部上拉电阻
    digitalWrite()    向数字引脚写入HIGH（5V）或LOW（0V）
        |-->第一参数：pin 引脚号
        |-->第二参数：HIGH或LOW
    digitalRead()   读取数字接口的值，并将该值返回
        |-->第一参数：pin 引脚号
    analogWrite()    给一个模拟接口写入模拟量（PWM脉冲）
        |-->第一参数：pin 引脚号
        |-->第二参数：数值
    analogRead()    检测是否有一个电压施加到其引脚，并读取其值
        |-->第一参数：pin 引脚号
        此函数返回0到1023之间的数字，表示0到5伏特之间的电压

时间函数
    delay()   延时一段时间  单位：毫秒
        |-->第一参数：时间
    delayMicroseconds()   延时一段时间  单位：微秒 
        |-->第一参数：时间    
    millis()   返回从开始运行当前程序时的毫秒数
字符函数
    int isdigit(int c)     如果c是数字，则返回1，否则返回0
    int isalpha(int c)     如果c是字母，则返回1，否则返回0
    int isalnum(int c)     如果c是数字或字母，则返回1，否则返回0
    int isxdigit(int c)    如果c是十六进制数字字符，则返回1，否则返回0
    int islower(int c)     如果c是小写字母，则返回1，否则返回0
    int isupper(int c)     如果c是大写字母，则返回1；否则返回0
    int isspace(int c)     如果c是空白字符：换行符（'\n'）、空格符（' '）、换页符（'\f'）、回车符（'\r'）、水平制表符（'\t'）
                           或垂直制表符（'\v'），则返回1，否则返回0
    int iscntrl(int c)     如果c是控制字符，如换行符（'\n'）、换页符（'\f'）、回车符（'\r'）、水平制表符 （\v'）、垂直制表符（'\v'）、
                           alert（'\a'） 或退格（'\b'），则返回1，否则返回0
数字库
    max(x,y)    返回两者中的最大值
    min(x,y)    返回两者中的最小值
    abs(x)      返回该参数的绝对值
    sq(x)       返回该参数的平方
    pow(base,exponent)    返回base的exponent次方
    sqrt(x)               返回该参数的平方根
    constrain(x,a,b)      防止数字x溢出a~b的范围
    random()    随机返回一个整数
通信库
    UART串口
        Serial.begin(9600);    配置串口UART的波特率为9600
        Serial.available();    检测串口是否可用
        Serial.print("xxxx");  串口将发回实际的ASCII代码
        Serial.write();        串口将发送实际的文本
        Serial.read();         读取串口接收到内容
    IIC
        Wire.begin(地址)        配置本板IIC的地址
        Wire.beginTransmission(地址)       开始向给定地址的I2C从设备发送数据
        Wire.write(值)          用于从主设备传输到从设备的队列字节（在beginTransmission()和endTransmission()之间的调用）
        Wire.endTransmission()  结束由beginTransmission()开始的对从设备的传输，并传输由wire.write()排队的字节

        Wire.onReceive(收到的数据处理程序)   当从设备从主设备接收数据时调用的函数
        Wire.available()                    返回Wire.read()可用于检索的字节数，应在Wire.onReceive()处理程序中调用
    SPI
        SPI.begin()     通过将SCK，MOSI和SS设置为输出来初始化SPI总线，将SCK和MOSI拉低，将SS拉高
        SPI.setClockDivider(分频器)     相对于系统时钟设置SPI时钟分频器
        SPI.transfer(val)      SPI传输基于同时发送和接收：接收的数据在receivedVal中返回
        SPI.beginTransaction(SPISettings(speedMaximum，dataOrder，dataMode)) 
                        speedMaximum是时钟，dataOrder(MSBFIRST或LSBFIRST)，dataMode(SPI_MODE0，SPI_MODE1，SPI_MODE2或SPI_MODE3
中断
    Interrupt()   可以被中断
    noInterrupt()    不可以被中断
        外部中断：
              attachInterrupt(中断号,中断服务函数入口, 触发类型);
                |--> LOW  低电平触发
                |--> CHANGE  输入发生改变时触发
                |--> RISING  上升沿触发
                |--> FALLING 下降沿触发
              detachInterrupt(中断号)    取消中断函数
        定时器中断：
              void set(unsigned long ms,void(*f)())       设置定时中断函数
                        ms：定时时间的间隔长度
                        void(*f)()   调用的中断服务函数入口
              void start()     开启定时中断函数
              void stop()      关闭定时中断函数
    <定时器中断函数都是在MsTimer2的作用域下>
        eg:
            #include "MsTimer2.h"
            MsTimer2 :: set(500,blink);
            MsTimer2 :: start();

            void blink()
            {
                ......
            }
舵机
    servo.attach(pin)     告知Arduino舵机的数据线连接在哪一个引脚（pin）上
    servo.write(angle)    将舵机轴旋转到相应的角度位置
    servo.read()          获取当前舵机轴角度信息
